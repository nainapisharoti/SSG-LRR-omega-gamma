# SSG-LRR-omega-gamma
RST-based transition turbulence model, SSG/LRR-omega-gamma, implemented on OpenFOAM-v2006. 
Model formulation and description can be found in:
Pisharoti, N., Webster, J., & Brizzolara, S. (2022). Reynolds Stress Turbulence Modelling with Î³ Transition Model. International Journal of Computational Fluid Dynamics, 1-23. DOI: https://doi.org/10.1080/10618562.2022.2070614
Kindly cite the above publication when using the code or the turbulence model.
